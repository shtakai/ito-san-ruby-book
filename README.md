# ito-san-ruby-book
伊藤さんのチェリー本を章ごとにgitのブランチで切って、githubでプルリクエストしよう。
rspecに慣れるためにもminitestと並行して、rspecでも書いてみよう

# 注意
- forkはめんどくさいからだめかな
- cloneしても参照用にするか、リモートを自分のリポジトリ作ってそこに変えたほうがいい
- 最初の履歴いらんのなら、zipをDLしてgit initするのがいい
